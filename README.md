A hyper-local, real-time, collaborative platform for citizen journalism.

Work-in-progress, more coming soon!

## Usage

### Requirements:

- Python 3
- Django

Clone repo. Run:

```bash
python manage.py runserver
```

Browse to `localhost:8000`.
